# Aperture
 
A plain-language database of research, internship, and fellowship opportunities for first-generation and non-target students. Everything that exists, clearly described, in one place.
 
---
 
## What it does
 
- Curated database of REUs, fellowships, internships, and research programs
- Plain-language descriptions — no jargon, no vague eligibility language
- AI-assisted matching based on background, interests, and goals
- Peer mentorship layer connecting students who found opportunities with those still looking (coming soon)
## Tech stack
 
- **Frontend** — Next.js, React, TypeScript, Tailwind CSS
- **Database** — Supabase (PostgreSQL)
## Getting started
 
### Prerequisites
 
- Node.js 18+
- Supabase account or local instance
### Install
 
```
git clone https://github.com/sunrise-atelier/aperture
cd aperture
npm install
```
 
### Run locally
 
```
npm run dev
```
 
Open [http://localhost:3000](http://localhost:3000).
